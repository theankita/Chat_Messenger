>Chat Messenger
A simple Chat Messenger Application built using Java.
This project demonstrates basic messaging functionality between users using GUI and networking concepts.
The application is designed for learning purposes and shows how a chat system works using client-server communication, which is a common approach in messenger apps.

>Features
Send and receive messages
Simple GUI interface
Real-time chat communication
Client-Server connection
Lightweight desktop application

>Tech Stack
Java
Swing / JFrame (GUI)
Socket Programming
VS Code

>Purpose of Project
This project was created to learn:

Java networking
GUI development
Client-Server architecture
Real-time communication
